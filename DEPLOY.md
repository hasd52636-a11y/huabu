# 🚀 曹操画布工作站 - 部署指南

## 快速部署

### 方法1: Vercel CLI (推荐)
```bash
# 1. 构建项目
npm run build

# 2. 部署到Vercel
vercel --prod
```

### 方法2: 手动部署
```bash
# 1. 运行部署脚本
deploy-production.bat

# 2. 上传dist文件夹到服务器
```

## 环境配置

### 必需环境变量
```env
GEMINI_API_KEY=your_production_api_key
NODE_ENV=production
```

### 重要要求
- ✅ **HTTPS必需** (摄像头/麦克风权限)
- ✅ **现代浏览器** (Chrome 80+, Firefox 75+, Safari 14+)
- ✅ **API密钥配置** (Google Gemini)

## 核心功能

✅ **曹操AI助手** - 语音唤醒"曹操"
✅ **手势控制系统** - 摄像头手势识别
✅ **语音控制系统** - 智能语音指令
✅ **智能画布** - 多模态内容创作
✅ **实时反馈** - 音效和视觉提示

## 使用说明

### 激活多模态控制
1. 打开右侧"曹操"标签页
2. 点击"语音控制"或"手势控制"按钮
3. 允许摄像头/麦克风权限

### 语音控制
- 说"曹操"唤醒AI助手
- 自然语言指令："生成文字"、"画图片"等

### 手势控制
- 键盘快捷键测试：Z/X缩放，方向键移动
- 实际手势识别（摄像头）
