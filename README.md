# 曹操画布 - 轻量级自动化多媒体工作站

说曹操，曹操到——这里是曹操画布，您的专业多媒体创作伙伴！💫

曹操画布是一个基于AI的智能创作平台，支持文本、图片、视频的生成和编辑。用户可以通过模块化的方式创建内容，并通过连接模块实现自动化工作流。

## 核心特性

- 🤖 **AI增强手势控制** - 智能识别手势，自然操控画布
- 🎤 **语音指令系统** - 说"曹操"唤醒，语音控制全流程
- 📝 **模块化创作** - 文本、图片、视频模块化生成
- 🔗 **智能连接** - 模块间自动化数据传递
- 🎨 **多模态AI** - 支持多种AI服务提供商
- 📱 **响应式设计** - 适配各种设备和屏幕

## 本地运行

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
